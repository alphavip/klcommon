- gdb filename : load the dest file but donot run it
- r arguments : run the program and pass the arguments to the program
- break func : set a break point at the function
- clear : clear the specified break points
- bt : to display the call stack
- info args : to display the arguments of the function 
- info locals : to display the local variables of the function(current fream)
- l : to list specified function or line ( list source code )
- n : to execute the next line when the program breaks
- s : to step in the function, different from 'n' 
- c : continue to run the program
- p : to print some variables value
- display : to display some variables value automatically when the program stops
- finish : to execute until the function returns
- break address if condition : break if the condition is true
